What is the profitability index calculator?
The profitability index calculator is a tool that calculates the potential profitability of a project or investment.
How to calculate profitability index?
To calculate profitability index you should use the following formula:
profitability index = pV of future cash flow / initial investment
Why use profitability index calculator?
The profitability index calculator is used in order to determine the potential profitability of a project or investment.
